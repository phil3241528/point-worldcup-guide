# point-worldcup-guide

## 项目简介

`point-worldcup-guide` 是一个用于归档和发布多个独立 HTML 页面的仓库。这些页面以静态形式存放，不绑定任何特定域名或网站，旨在提供一种轻量、可访问的文档化浏览方式。

本仓库适合存放与“世界杯”主题相关的参考指南、信息汇总或工具页面，但内容不限于此，任何需要以独立 HTML 页面形式发布的资料均可纳入。

## 目录结构

```
point-worldcup-guide/
├── index.html          # 仓库首页/导航页
├── pages/              # 存放所有独立 HTML 页面
│   ├── example.html
│   └── ...
├── assets/             # 共用的静态资源（CSS、图片、字体等）
│   ├── css/
│   ├── images/
│   └── ...
└── README.md           # 本文件
```

- `index.html`：可选，用于提供页面列表或导航链接。
- `pages/`：核心目录，每个 HTML 文件为一个独立页面，可直接通过文件路径访问。
- `assets/`：存放页面间共用的样式、图片或其他资源。

## 页面归档说明

- 所有页面均为纯 HTML 格式，不依赖后端服务。
- 页面之间相互独立，无强制关联，可单独打开使用。
- 页面内容保持稳定，不主动更新，除非有重大修正需求。
- 归档页面不包含跟踪代码、广告或外部依赖，以保障隐私与可移植性。

## 维护说明

- 本仓库接受与“世界杯”主题相关的指南、信息汇总或工具类 HTML 页面贡献。
- 新增页面请放入 `pages/` 目录，并在 `index.html`（若存在）中添加链接。
- 请确保页面内不包含任何具体域名、网址、营销承诺或诱导点击内容。
- 提交前请检查 HTML 语法及资源引用路径是否正确。
- 欢迎通过 Issue 或 Pull Request 提出改进建议或报告问题。
